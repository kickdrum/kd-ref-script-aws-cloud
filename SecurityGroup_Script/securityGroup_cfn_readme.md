# Cloudformation Security Groups

## Introduction
This is the Cloud formation template to configure two types of Security Groups


## Resources
The major resources it sets up are - 
Security Group
- 1 Security group
- Inbound rule
- Outbound rule
- Inbound/Outbound rule referencing another security group
