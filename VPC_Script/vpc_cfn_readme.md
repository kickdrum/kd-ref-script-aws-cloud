# Cloudformation VPC

## Introduction
This is the Cloud formation template to configure a VPC with different parameters

## Resources
The major resources it sets up are -
- 1 VPC
- N public subnets
- N private subnets
- Internet Gateway
- NAT Gateway (1 , 2-for high availability)
- Route Table, Routes
