# Cloudformation S3

## Introduction
This is the Cloud formation template to configure a S3 Bucket with different parameters

## Resources
The major resources it sets up are -
- 1 S3 Bucket
- Encrypted at rest using KMS key
- Logging enabled
- Versioning enabled
- Lifecycle configuration
